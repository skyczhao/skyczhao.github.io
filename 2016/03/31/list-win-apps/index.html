Windows下常用软件列表

Mar 31, 2016

系统工具

  1. EasyBCD
    多系统启动管理
  2. VirtualBox

开发工具

  1. xshell
  2. putty & winscp
  3. Visual Studio
  4. Photoshop
  5. SmartGit
    如果需要SSH Keys，可以利用上面的GitHub for Windows，它带有Git Shell
    然后使用ssh-keygen.exe -t rsa -C "example@email.com"生成key， 默认在C:\Users\[用户名]\.ssh中可以找到id_rsa以及id_rsa.pub
  6. JetBrains
    • PyCharm
    • IntelliJ IDEA
  7. Github
  8. JDK
  9. python
  10. TortoiseSVN

文本工具

  1. notepad++
    小巧的编辑器，加上zen coding插件后简直神器。
  2. MarkdownPad
    Markdown 编辑器
  3. Microsoft Office
  4. Adobe Reader

网络通信

  1. QQ
  2. Skype
  3. Chrome
  4. Firefox
  5. Flash
  6. RealVNC

影音娱乐

  1. QQ影音
  2. foobar2000
  3. Picasa
    看图，Google家的应用
  4. 迅雷
  5. utorrent
  6. PowerISO

日常使用

  2. 搜狗输入法
